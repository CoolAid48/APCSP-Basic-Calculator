## APCSP-Basic-Calculator
**This repo contains the source code for my basic calculator program for my AP Computer Science Principles Create PT**

The AP Computer Science Principles (CSP) Create Performance Task (PT) is a major, required project accounting for 30% of the final exam score where students design, code, and document a custom computer program to solve a problem or express creativity. It entails submitting a video, program code, and a personalized reference to be discussed in a written exam

- [College Board Create Performance Task PDF](https://apcentral.collegeboard.org/media/pdf/ap-csp-student-task-directions.pdf)
